# Smart-Drug-Tracking-Web-app
A modern, glass-UI hospital drug inventory system with role-based access, IoT temperature simulation, FEFO-based expiry tracking, real-time alerts, and AI-style demand forecasting.
Fully front-end (HTML/CSS/JS) and runs instantly in any browser using localStorage.
